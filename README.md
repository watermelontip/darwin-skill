# 达尔文视角技能 (Darwin Perspective Skill)

## 简介

这是一个让AI助手以查尔斯·达尔文的思维方式来分析问题的技能。当你需要一种冷静、耐心、基于事实的视角来思考复杂问题时，可以触发这个技能。

## 使用方式

在对话中使用以下触发词即可激活：

- **达尔文**
- **用达尔文的方式**
- **从达尔文的角度**
- **达尔文怎么看**

例如：
- "从达尔文的角度来看，这个创业项目该怎么评估？"
- "用达尔文的方式思考一下这个技术选型"
- "达尔文怎么看当前的AI竞争？"

## 核心思维模型

| 模型 | 核心要点 |
|------|----------|
| 自然选择 | 不是最强的生存，是最适应的 |
| 长期主义 | 好东西需要时间，不急于求成 |
| 观察-归纳法 | 先收集事实，再归纳规律 |
| 变异与选择 | 错误是变异的来源，试错是进步的方式 |
| 连接跨领域 | 最好的想法来自跨领域碰撞 |
| 质疑权威 | 用事实说话，不盲从 |

## 适用场景

- 需要冷静分析复杂局面时
- 面对不确定性需要决策时
- 需要长期视角而非短期思维时
- 需要跳出常规思维寻找创新方案时
- 需要对抗焦虑、保持耐心时
